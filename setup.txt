laravel new hypersender-challenge
composer require filament/filament:"^3.0" -W
php artisan filament:install --panels
php artisan make:filament-user

------------------------ Start in creating the filament resources ----------------------

1- php artisan make:filament-resource {ModelName} - in my case ->Company
1.1- php artisan make:filament-relation-manager CompanyResource drivers name
1.2- php artisan make:filament-relation-manager CompanyResource vehicles plate_number
1.3- php artisan make:filament-relation-manager CompanyResource trips
==== php artisan make:exception OverlappingTripException (to use it in the relationManager to return filament notification.)

2- php artisan make:filament-resource Driver --generate ( to generate the model attributes in the schema and table by default)
2.1- php artisan make:filament-relation-manager DriverResource trips

3- php artisan make:filament-resource Vehicle --generate
3.1- php artisan make:filament-relation-manager VehicleResource trips

4- php artisan make:filament-resource Trip --generate

5- php artisan make:filament-widget ActiveTripsCount --stats-overview => blank => admin
6- php artisan make:filament-page --type=dashboard Dashboard
7- php artisan make:filament-widget AvailableDriversKPI --stats-overview => blank => admin
8- php artisan make:filament-widget AvailableVehiclesKPI --stats-overview => blank => admin
9- php artisan make:filament-widget CompletedTripsKPI --stats-overview => blank => admin

10- php artisan pest:test TripTest
11- php artisan pest:test KPIWidgetTest

------ php artisan test --filter={filename} =>> command to run specific test file -------------
